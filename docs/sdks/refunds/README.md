# Refunds
(*Refunds*)

## Overview

### Available Operations

* [Get](#get) - Get refund

## Get

Fetch a refund.

### Example Usage

```csharp
using gr4vy;
using gr4vy.Models.Components;

var sdk = new Gr4vy(
    bearerAuth: "<YOUR_BEARER_TOKEN_HERE>",
    merchantAccountId: "default"
);

var res = await sdk.Refunds.GetAsync(
    refundId: "6a1d4e46-14ed-4fe1-a45f-eff4e025d211",
    merchantAccountId: "default"
);

// handle response
```
